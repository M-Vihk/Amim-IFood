# Amim-IFood
A aplicação Amim iFood é um protótipo de e-commerce de alimentos com design em tons de rosa, oferecendo uma navegação simples e intuitiva. Possui telas de home, carrinho, contato e perfil do usuário, com carrossel de destaques, cards de produtos e barra fixa, tornando o fluxo de compra prático e agradável.
